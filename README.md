# Foundryvtt-D6Space

A FoundryVTT system for playing OpenD6 Space, forked from [opend6-space](https://gitlab.com/vtt2/opend6-space).

Uses the rules from the OpenD6 Project SRD found at http://opend6project.org. Content comes from the OpenD6 Space System. The ruleset has been released under the terms of the OGL v1.0a.
